***************BASIC HTML ****************
1. <!DOCTYPE html> :- Defines the document type 
2. <html></html> :- Defines an HTML document
3. <head></head> :- Contains metadata / information for the document
4. <title></title> :- Defines a title for the document
5. <body></body> :- Defines the document`s body 
6. <h1> -> <h6> :- Defines HTML headings
7. <p></p> :-  Defines a paragraph
8. <br> :- Insert a single line break
9. <hr> :- Defines a thematic change in the content 
10. <!-- --> :- Defines a comment
